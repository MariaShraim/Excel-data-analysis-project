Bike Buyers Data Analysis Project
Overview

This project was created in Microsoft Excel as part of a data analytics practice project.

The project focuses on analyzing customer data related to bike purchases using Excel data analysis techniques.

Project Steps

1. Data Cleaning

The raw dataset was cleaned and organized using:

Duplicate checking
Formatting adjustments
Excel formulas and functions
Data standardization

2. Pivot Tables

Pivot tables were created to analyze:

Customer income
Age groups
Commute distance
Bike purchase patterns

3. Dashboard Creation

An interactive dashboard was built using charts and visualizations generated from the pivot tables.
